# Gemini-Blueprint
⚡ Landing page of a "Gemini Course" built using Gemini (Canvas tool + Pro)

💻 Prompt Used -

You're an expert web designer. Create a high-converting HTML landing page for an AI course titled 'The Gemini Blueprint'. Include a catchy headline, placeholders for 4 different live sessions, each with 3 bullet points of what you'll learn, a countdown timer to release date, and a simple email signup form.
